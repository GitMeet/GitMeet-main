GitMeet

GitMeet is a platform which connects open source project creators with developers. GitMeet enables developers to locate projects and begin active contribution.

Create projects and invite developers, or find a project to join with GitMeet

Build On Repositories From Github

Create projects by importing current repositories from Github and allowing GitMeet to format a custom project. Users can then request to join the project or be invited by the project creator

Search Projects

Browse projects posted and send custom project join requests. Subscribe to receive emails with recommended projects that match your personal Github projects and listed skills.

Receive Developer Recommendations

Struggling to find team members? Enable developer recommendations to receive emails with listings of developers whose skills match those listed on your project.
