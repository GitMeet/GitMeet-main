GitMeet

GitMeet is an platform enabling project creators to better attract and retain repository contributors. GitMeet provides options to invite developers, award contributions, and expand a project's talent base.

Goals

  • Increase the amount of beneficial project contributions
  • Retain contributors throughout project development
  • Better organize, categorize, tag, and manage project issue and job postings

For Project Creators

GitMeet provides project creators simple, powerful tools to better manage project issue and job postings, along with user join requests.

For Project Teammembers
